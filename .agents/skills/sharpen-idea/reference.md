# 参考资料:评估表格与提问话术

来源:《初级二:找到好点子》方法论。供 sharpen-idea 各步骤查阅。

**注意:本文件中的所有示例(尤其记账APP→留学生家长案例)仅作格式参考,不得把示例结论直接喂给用户——人群和场景必须由用户真实推导得出。**

## 1. 用户访谈提问话术(Step 3 / 交棒前)

**禁问:**"你会用我的产品吗?"(假阳性率 90%)

**要问(行为提问法):**

| 问题 | 目的 |
|------|------|
| "你现在怎么解决这个问题?" | 了解真实行为 |
| "最近一周,这个问题困扰了你几次?" | 了解频率 |
| "为了解决它,你花了多少钱/时间?" | 了解付费意愿 |
| "如果有解决方案,但要改变习惯,你愿意吗?" | 了解改变成本 |
| "如果收费 XX 元,你会买吗?" | 了解价格敏感度 |

判断标准:
- 3 个以上用户说"我每天都在为这事头疼" → 可能是痛点
- "挺有意思,但我不着急" → 大概率是痒点
- "我现在用 XX 解决,但不太满意" → 有机会

## 2. 替代方案机会判断表(Step 2)

| 用户现在的替代方案 | 机会判断 |
|--------------------|----------|
| 没有替代方案,默默忍受 | 大机会,但需要教育市场 |
| 很笨的方法(Excel、手工、靠记忆) | 好机会,用户渴望更好的方案 |
| 多个工具拼凑 | 好机会,整合有价值 |
| 成熟产品但不满意 | 有机会,需要差异化 |
| 成熟产品且很满意 | 机会很小,除非颠覆式创新 |

## 3. 人群评估维度(Step 2)

每个细分人群按 5 个维度打分(高/中/低):

1. **痛点强度**:是痛点还是痒点?不解决的后果多严重?
2. **付费意愿**:愿意为解决方案付多少钱?
3. **市场规模**:目标用户数 × 付费意愿 × 客单价
   - >10亿:大赛道 | 1-10亿:中小赛道 | <1亿:小众/副业
4. **竞争程度**:现有方案是否令人满意?
5. **你对人群的理解**:是否身处其中或有接触渠道?

示例(记账 APP):

| 人群 | 痛点 | 付费意愿 | 规模 | 竞争 |
|------|------|----------|------|------|
| 普通上班族 | 记录麻烦 | 低 | 大 | 高 |
| 小微企业主 | 公私支出混淆 | 高 | 中 | 中 |
| 自由职业者 | 现金流难预测 | 高 | 中 | 中 |
| 留学生家长 | 孩子花钱不透明 | 高 | 小 | 低 |

→ 选留学生家长:痛点最痛、竞争最低。

## 4. 场景深挖示例(Step 3)

人群:产后妈妈

| 时间 | 场景 | 痛点 | 情绪 |
|------|------|------|------|
| 早上6点 | 宝宝刚睡,有30分钟 | 不知道做什么动作安全 | 恐惧 |
| 上午10点 | 抱娃哄睡,腰酸 | 手没空,想做修复运动 | 焦虑 |
| 下午3点 | 宝宝睡了,想运动 | 身体累,不知道还能不能做 | 无助 |
| 晚上8点 | 终于有空 | 看镜子觉得人生完蛋了 | 抑郁 |
| 长期 | 没人理解 | 觉得只有自己这么痛苦 | 孤独 |

重构:从"健身工具"升级为"康复教练 + 心理支持者"。

## 5. 价值重构案例库(Step 4)

| 原始想法 | 细分人群 | 真实痛点 | 重构后 |
|----------|----------|----------|--------|
| 健身 APP | 产后妈妈 | 恢复焦虑、无人指导 | 产后妈妈恢复助手 |
| 卡路里拍照 | 糖尿病患者 | 外食碳水难估算,怕血糖飙升 | 糖友安心吃 |
| 新闻聚合 | 金融分析师 | 行业信息分散,追踪低效 | 投研情报官 |
| 记账 APP | 留学生家长 | 孩子海外消费失控感 | 留学资金管家 |
| 番茄钟 | 远程工作者 | 老板看不见,信任危机 | 远程工作证明 |
| 二手书平台 | 宝妈 | 绘本生命周期短,买新太贵 | 绘本图书馆订阅 |
| 校园二手平台 | 毕业生 | 离校前一周清不完东西 | 毕业清仓助手 |

共同模式:**功能 → 某个具体人群的情绪解决方案**。

## 6. 护城河类型与验证红线指标

护城河类型(Step 4 输出"竞争壁垒"时参考;早期项目可以没有,但要想清楚方向):网络效应、数据积累、品牌认知、规模效应、技术专利、转换成本。

定金/预售转化率(Step 5 交棒时引用):
- **>10%**:需求真实,值得投入
- **5-10%**:需求存在,需要打磨
- **<5%**:需求不成立,调整方向

## 7. 记忆口诀
